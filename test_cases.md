**Позитивный сценарий**

ID: ТС01
  
Название: Успешная оплата банковской картой
  
Предусловия: 
  1. Пользователь авторизован в приложении
  2. В корзину добавлен товар
  3. открыта страница для оплаты товара
  
Шаги:
  1. на странице оплаты в поле Номер банковской карты ввести: 2234 1234 1234 1234
  2. в поле Срок действия ввести: 12.03.2030
  3. в поле CVV ввести: 123
  4. нажать на кнопку Оплатить 
  5. при необходимости пройти 3-D Secure
  
Ожидаемый результат: 
  1. Данные успешно отправлены в платежный шлюз
  2. Платёж получает статус "Успешно"
  3. Статус заказа обновляется на "Оплачен"
  4. Пользователю отображается сообщение об успешной оплате.
  5. На email отправлен чек.
